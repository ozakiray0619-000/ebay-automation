## 補足: キーを受け取ったあとの設定先

GAS エディタ → 歯車（プロジェクトの設定）→ スクリプトプロパティ:

| プロパティ名 | 値 |
|---|---|
| `ANTHROPIC_API_KEY` | 受け取ったキー（`sk-ant-...`） |

これを入れると、デモ実行時に実際に Claude が返信案を生成します。
未設定でもデモのフロー自体は流れます（返信案部分が「未設定のためスキップ」表示になるだけ）。

## キーが間に合わない場合のデモ代替案

5/21 までにキーが届かない場合でも、デモは成立させられます:

1. **フロー実演**: モック実行で「メッセージ取得 → 記録 → Discord通知」までは動く（返信案だけプレースホルダ表示）
2. **返信案サンプルを事前生成**: 私（このチャット）でモック3件の返信案を先に作っておき、デモ時に「実際にはこういう文面が出ます」と提示

→ 必要なら「返信案サンプルを作って」と言ってください。3件分すぐ出します。
