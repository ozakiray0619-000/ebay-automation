## 補足: 各項目の意図（自分用メモ）

| 項目 | なぜ必要か | コードへの影響 |
|---|---|---|
| 1. マーケットプレイス | Site ID が変わる | `Config.gs` の `TRADING_API_SITE_ID` 確定 |
| 2. アカウント種別 | 取れる scope が変わる | Store契約有無で `sell.fulfillment` の挙動差 |
| 3. Best Offer 有効化 | F-03 の動作確認可否 | 未使用なら検証用商品で先に有効化依頼が必要 |
| 4. Discord URL | 本番通知先 | `DISCORD_WEBHOOK_URL` スクリプトプロパティ |
| 5. OAuth 同意 | Phase 3 必須 | 同意後、Web App URL を送って認可リンクをクリックしてもらう |

回答が揃った時点で、Phase 3（本番移行）の準備が完了します。
